El juego consiste en lanzar una pelota para atravesar obstaculos y llegar al final del nivel.

Esta implementado el uso de fisicas, joints, colliders y el tiro parabólico. Ademas se puede visualizar la trayectoria de la bola.

El juego cuenta con menú y gameplay. Consitiendo este último de tres niveles.
